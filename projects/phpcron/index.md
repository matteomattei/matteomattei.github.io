PhpCron is a library for running and managing cron jobs without a real cron daemon (useful for Linux web hostings that donâ€™t allow cron jobs).

Technologies used:
------------------
 - PHP 5
 - INI file format

Requirements:
-------------
 - Linux hosting

Main Features:
--------------
 - External ini file to customize and configure cronjobs
 - Possibility to run different types of interpreters
 - Check for single instance of the program
 - Limited support for linux/unix cronjob syntax
